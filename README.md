# CloudNest

CloudNest is a secure cloud-based file management web application designed to help users upload, organize, manage, and share files efficiently through an intuitive and responsive user interface.



## Features

* User authentication (Sign In / Sign Up)
* File upload and management
* Drag and drop file upload functionality
* Folder creation and organization
* Trash management (restore deleted files)
* File and folder sharing
* Subscription and plan management
* User-friendly dashboard
